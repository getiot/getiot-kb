# Ubuntu 简介



Ubuntu 是一个基于 Debian 架构的 Linux 发行版。它可以免费下载和使用，并且开源，因此可以根据用户的应用进行更改。Ubuntu 系统包含超过上千款开源软件，包括 Linux 内核、Shell 终端、GNOME/KDE 桌面环境、标准桌面应用（包括文字处理软件、网页浏览器、电子表格软件、Web 服务器、编程语言、集成开发环境 IDE 和一些计算机游戏等）。

Ubuntu 系统支持 Intel x86、AMD-64、ARMv7 和 ARMv8（ARM64）等处理器架构，可以运行在台式机、服务器，以及嵌入式设备上（如树莓派）。目前，Ubuntu 由一家英国公司 Canonical Ltd. 负责维护。
