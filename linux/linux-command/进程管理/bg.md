# Linux bg 命令 - 将作业置于后台运行



## 介绍

**bg** 命令用于将作业放到后台运行，使前台可以执行其他任务。该命令的运行效果与在指令后面添加符号 `&` 的效果是相同的，都是将其放到系统后台执行。与 [fg](/linux-command/fg) 命令一样，若任务列表中只有一个，则使用该命令时，可以省略任务号。

提示：程序正在前台运行，可以使用 `Ctrl + Z` 把程序暂停，`Ctrl + C` 终止程序。
