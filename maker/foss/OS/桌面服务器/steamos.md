# SteamOS 操作系统

![SteamOS 操作系统](https://static.getiot.tech/steamos-banner.png)

**SteamOS** 是 Valve 基于 Linux 的操作系统。该系统针对游戏方面进行了优化，可提供无缝的用户体验，同时保留了 PC 的强大功能和灵活性。SteamOS 可以畅玩 Steam 上成千上万的游戏，并且由于底层是开放的 Linux 平台，因此你拥有完全控制，可以随心所欲地安装新的软件或内容。默认情况下，Steam 客户端将作为用户界面存在，并用来连接至 Steam 在线服务，但你仍可使用标准的 Linux 桌面。

:::tip

Valve（维尔福）是一家美国视频游戏开发商、发行商和数字发行公司，总部位于华盛顿州贝尔维尤市。该公司开发了数字发行平台 Steam，并打造了《半衰期》《反恐精英》《传送门》《胜利之日》《军团要塞》《求生之路》和《DOTA 2》等知名游戏系列。

:::



## SteamOS 核心功能

SteamOS 可以让用户访问整个 Steam 生态系统：Steam 库、Steam 云、Steam 聊天、Steam 游戏录制，以及更多其他功能。用户界面经过优化，便于控制器的使用，并提供快速暂停/继续等功能，让玩家能够迅捷地进入或离开游戏。最棒的是，用户无需担心驱动程序更新和硬件调整 —— 所有这些都会通过内置的系统更新无缝处理。



## SteamOS 游戏兼容性

SteamOS 是基于 Arch Linux 开发的，而得益于 Valve 提供的 Proton 兼容层，Steam 库中的绝大多数游戏都可以在 SteamOS 上运行。这也使得开发者可以发布适用于 Windows 的游戏，无需担心将其移植到 Linux 的问题，游戏也能 SteamOS 上无缝运行。如果游戏无法在 SteamOS 上顺畅运行，我们会通过“Deck 验证”项目将其指出，从而让顾客知晓哪些游戏运行良好，哪些游戏运行不畅。
