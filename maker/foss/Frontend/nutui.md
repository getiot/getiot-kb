# NutUI 京东开源的移动端组件库

NutUI 是京东零售开源的一套京东风格的移动端组件库，使用 MIT 许可证发布。NutUI 项目专为移动端 H5 和小程序开发场景而设计，目前包含 Vue 组件库和 React 组件库两大分支，内含近百个高质量组件，支持按需引用、TypeScript、国际化等特性。



## 主要特性

- 🚀 80+ 高质量组件，覆盖移动端主流场景
- 💪 支持一套代码同时开发 H5+多端小程序
- 📖 基于京东APP 10.0 视觉规范
- 🍭 支持按需引用
- 📖 详尽的文档和示例
- 💪 支持 TypeScript
- 💪 支持服务端渲染（测试阶段）
- 🍭 支持组件级别定制主题，内置 700+ 个变量
- 🌍 国际化支持，已支持英文，印尼语和繁体中文
- 🍭 单元测试覆盖率超过 80%，保障稳定性
- 📖 提供 Sketch 设计资源
